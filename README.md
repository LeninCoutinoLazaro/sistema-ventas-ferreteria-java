# Sistema gestor de ventas

Un sistema gestor de ventas orientado a una ferretería desarrollado en el entorno de desarrollo Netbeans, en el lenguaje de Java e integrando la tecnología RMI.

## Descripción
Este proyecto consiste en el desarrollo de un sistema gestor de ventas diseñado para administrar las operaciones básicas de una ferretería. El sistema permite registrar productos, controlar inventario, realizar ventas y consultar información relacionada con los movimientos comerciales del negocio.

El sistema fue desarrollado como parte de un proyecto académico durante la formación en Ingeniería en Informática, con el objetivo de aplicar conceptos de programación orientada a objetos, gestión de bases de datos y desarrollo de interfaces de usuario.

## Tecnologías utilizadas

El desarrollo del sistema se realizó utilizando las siguientes tecnologías:
- Lenguaje de programación: Java
- Base de datos: MySQL
- Entorno de desarrollo: NetBeans
- Paradigma de programación: Programación orientada a objetos

## Funcionalidades

El sistema incluye las siguientes funcionalidades principales:

- Registro y gestión de productos.
- Administración del inventario disponible.
- Registro de ventas realizadas.
- Consulta de información sobre productos y ventas.
- Interfaz gráfica para facilitar la interacción del usuario con el sistema.

## Estructura del proyecto

El repositorio contiene un archivo comprimido (.rar) que incluye todos los archivos del proyecto, entre ellos:

- Código fuente del sistema.
- Archivos de configuración del entorno de desarrollo.
- Archivos relacionados con la conexión a la base de datos.
- Recursos utilizados en la interfaz del sistema.

## Cómo ejecutar el proyecto

Para ejecutar el sistema es necesario seguir los siguientes pasos:

1. Descargar el archivo comprimido disponible en este repositorio.
2. Descomprimir el archivo utilizando un software de descompresión como WinRAR o 7-Zip.
3. Abrir el proyecto desde el entorno de desarrollo correspondiente (NetBeans).
4. Configurar la conexión a la base de datos MySQL si es necesario.
5. Ejecutar el proyecto desde el entorno de desarrollo.
